The document.designMode property in JavaScript allows you to toggle a document (like a webpage) between a normal mode and an "editable" mode.

When document.designMode is set to 'on', the entire document becomes editable, which can be helpful for testing, debugging, or for creating rich-text editors.

      
        //Editable mode
        document.designMode = 'on';  
        

        //No editable mode
        document.designMode = 'off';
